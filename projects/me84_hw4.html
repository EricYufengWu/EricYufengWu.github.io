HW 4 - IoT Map Clock

Spring 2019

This is a map clock that points to the continent you select on a web-based dashboard. The robot is powered by an EV3 brick, which is programmed in labVIEW and connected to the internet. An IoT dashbord is set up in Systemlink cloud as the control panel of the clock. For each option, the motor is programmed to rotate a fixed distance and then back to the original position before its encoder gets reset.

Project partner: Molly Lipman, Tufts University '20
